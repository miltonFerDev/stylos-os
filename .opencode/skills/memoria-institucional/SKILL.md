# Memoria institucional

## Propósito

Transformar memoria personal en memoria organizacional registrando consultas, decisiones, aprendizajes y reuniones de forma estructurada y reusable.

## Reglas

1. **Nombre de archivo estándar.** Usar `YYYY-MM-DD-titulo-breve.md` para todos los registros en las cuatro subcarpetas.

2. **Cada registro incluye fecha y contexto.** Sin contexto, un registro es difícil de interpretar después.

3. **Decisiones: incluir opciones y resultados.** No basta con anotar qué se decidió. Registrar: contexto, opciones consideradas, decisión tomada, responsable, fecha de revisión y resultado posterior.

4. **Aprendizajes: validar antes de registrar.** Un aprendizaje no es una opinión. Debe estar respaldado por datos o resultados observados.

5. **Consultas: registrar la fuente de la respuesta.** La respuesta puede venir de datos, criterio interno, contador, IA, etc.

6. **Reuniones: incluir compromisos.** Cada acta debe listar los compromisos con responsable y fecha límite.

7. **Actualizar el README de cada subcarpeta** cuando se agregue un registro, para mantener el índice al día.
