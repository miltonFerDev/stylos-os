# Aprendizajes

## Propósito

Registrar lecciones validadas por la experiencia del negocio. Un aprendizaje no es una opinión, es una conclusión respaldada por datos o por resultados observados.

---

## Convenciones

- Nombre de archivo: `YYYY-MM-DD-titulo-breve.md`
- Cada aprendizaje debe incluir: qué pasó, qué se aprendió, cómo se valida y si cambió algún criterio o proceso.

## Formato

```markdown
# [Título del aprendizaje]

## Fecha
[YYYY-MM-DD]

## Qué pasó
[Evento o situación que generó el aprendizaje]

## Qué se aprendió
[La lección concreta]

## Cómo se valida
[Qué datos o evidencia respaldan este aprendizaje]

## Impacto
[Si este aprendizaje cambió algún criterio, proceso o decisión]

## Relevancia futura
[Cuándo puede ser útil este aprendizaje]
```
