# Reuniones

## Propósito

Documentar las reuniones relevantes del negocio: decisiones tomadas, compromisos asumidos, temas pendientes y próximos pasos.

---

## Convenciones

- Nombre de archivo: `YYYY-MM-DD-titulo-breve.md`
- Cada acta debe incluir: fecha, participantes, temas tratados, decisiones tomadas, compromisos y próxima reunión.

## Formato

```markdown
# [Título de la reunión]

## Fecha
[YYYY-MM-DD]

## Participantes
[Quienes asistieron]

## Temas tratados
1. [Tema 1]
2. [Tema 2]

## Decisiones tomadas
- [Decisión 1]
- [Decisión 2]

## Compromisos
| Quién | Qué | Fecha límite |
|-------|-----|-------------|
| [Nombre] | [Compromiso] | [Fecha] |

## Temas pendientes
- [Pendiente 1]
- [Pendiente 2]

## Próxima reunión
[Fecha y agenda preliminar]
```
