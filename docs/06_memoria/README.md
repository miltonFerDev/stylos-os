# Memoria institucional

## Propósito

Registrar consultas, decisiones, aprendizajes y reuniones para que el conocimiento del negocio no dependa de la memoria de una persona. Esta es la sección central de Stylos OS: transforma memoria personal en memoria organizacional.

---

## Subcarpetas

| Carpeta | Contenido |
|---------|-----------|
| `consultas/` | Consultas importantes realizadas y sus respuestas |
| `decisiones/` | Decisiones tomadas, con contexto, hipótesis y resultado |
| `aprendizajes/` | Lecciones validadas por la experiencia |
| `reuniones/` | Actas y notas de reuniones relevantes |

## Convenciones

- Cada registro se documenta en un archivo Markdown separado.
- Los nombres de archivo siguen el formato `YYYY-MM-DD-titulo-breve.md`.
- Cada registro debe incluir fecha, contexto, contenido y fecha de próxima revisión (si aplica).
- Las decisiones deben incluir: contexto, opciones consideradas, decisión tomada, responsable, fecha de revisión y resultado posterior.
- Los aprendizajes deben incluir: qué pasó, qué se aprendió, cómo se valida y si cambió algún criterio o proceso.
