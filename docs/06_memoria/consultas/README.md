# Consultas

## Propósito

Registrar consultas importantes realizadas sobre el negocio y sus respuestas. Una consulta es cualquier pregunta relevante cuya respuesta puede servir en el futuro.

---

## Convenciones

- Nombre de archivo: `YYYY-MM-DD-titulo-breve.md`
- Cada consulta debe incluir: fecha, pregunta, contexto, respuesta obtenida y fuente de la respuesta.

## Formato

```markdown
# [Título de la consulta]

## Fecha
[YYYY-MM-DD]

## Pregunta
[La consulta realizada]

## Contexto
[Por qué surge la pregunta, qué situación la genera]

## Respuesta
[La respuesta obtenida]

## Fuente
[De dónde salió la respuesta: análisis de datos, criterio interno, contador, etc.]

## Relevancia futura
[Por qué puede ser útil esta consulta en el futuro]
```
