# Decisiones

## Propósito

Registrar las decisiones importantes tomadas en el negocio, con su contexto, las opciones consideradas y los resultados posteriores. Esto permite aprender de lo hecho y explicar por qué se tomó cada decisión.

---

## Convenciones

- Nombre de archivo: `YYYY-MM-DD-titulo-breve.md`
- Cada decisión debe incluir: contexto, opciones consideradas, decisión tomada, responsable, fecha de revisión y resultado posterior.

## Formato

```markdown
# [Título de la decisión]

## Fecha
[YYYY-MM-DD]

## Contexto
[Situación que generó la necesidad de decidir]

## Opciones consideradas
1. [Opción 1]: [breve descripción]
2. [Opción 2]: [breve descripción]

## Decisión tomada
[Qué se decidió]

## Responsable
[Quién tomó la decisión]

## Justificación
[Por qué se eligió esa opción]

## Criterio aplicado
[Qué criterio interno respalda la decisión, si aplica]

## Fecha de revisión
[Cuándo se va a evaluar el resultado]

## Resultado posterior
[Se completa después de la revisión: funcionó, no funcionó, ajustes]
```
