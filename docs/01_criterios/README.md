# Criterios de gestión

## Propósito

Centralizar los criterios que guían las decisiones del negocio. Cada criterio debe ser explícito, accesible y revisable.

## Convenciones

- Cada criterio se documenta en un archivo Markdown separado dentro de esta carpeta.
- Un criterio debe incluir: descripción, justificación, condición de aplicación, fecha de creación y fecha de próxima revisión.
- Si un criterio cambia, se registra el cambio y la razón en `docs/06_memoria/decisiones/`.
- Los criterios son guías, no leyes inflexibles. Si una situación requiere excepción, se documenta la razón.
