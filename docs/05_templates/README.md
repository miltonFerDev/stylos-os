## Convenciones

- Cada plantilla se documenta en un archivo Markdown separado dentro de esta carpeta.
- Las plantillas son puntos de partida, no formularios rígidos. Se adaptan según la necesidad.
- Las plantillas deben ser simples y fáciles de completar. Si una plantilla genera fricción, se simplifica.
