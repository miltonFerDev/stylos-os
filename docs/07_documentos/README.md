# Documentos de referencia

## Propósito

Centralizar referencias a documentos importantes del negocio: legales, contables, de marca, reglamentos y otros. Los documentos sensibles no se almacenan en este repositorio, sino que se referencian con su ubicación en Google Drive u otro medio seguro.

## Convenciones

- No se almacenan documentos sensibles (DNI, contratos firmados, claves) en este repositorio.
- Se referencia la ubicación de cada documento (Google Drive, carpeta física, etc.).
- Si un documento tiene versión vigente, se indica cuál es.
