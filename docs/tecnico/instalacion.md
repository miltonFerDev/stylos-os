# Instalación

## Propósito

Instrucciones para instalar y configurar el entorno de trabajo de Stylos OS.

---

## Requisitos previos

- Git
- Editor de texto (VS Code recomendado)
- Acceso a Google Sheets
- Acceso a Google Drive

## Pasos de instalación

### 1. Clonar el repositorio

```bash
git clone <url-del-repositorio>
cd stylos-os
```

### 2. Configurar variables de entorno

```bash
cp .env.example .env
```

Completar los valores en `.env` con los datos reales. Ver `.env.example` para referencia.

### 3. Verificar estructura

PENDIENTE: Agregar comando de verificación de estructura.
