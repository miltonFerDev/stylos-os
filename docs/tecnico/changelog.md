# Changelog

## Propósito

Registrar los cambios relevantes del proyecto Stylos OS. No es un log de commits de Git, sino un resumen de los cambios que afectan cómo se usa o entiende el sistema.

---

## Formato

Cada entrada sigue el formato:

```
## [YYYY-MM-DD] Título del cambio

- **Tipo**: agregado / cambiado / corregido / eliminado
- **Descripción**: qué cambió y por qué
- **Impacto**: qué afecta este cambio
```

---

## Entradas
